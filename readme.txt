Lancement du serveur:
    Aller dans le dosser serveur
        lancer avec la commande : ant run

Lancement d'un client :
    Aller dans le dossier client dans un terminal
        Avant la première utilisation compiler avec la commande : make
        lancer avec la commande : ./client

Remarque :
    L'adresse du serveur peut être changé à la ligne 23 du fichier "build.xml". 
Elle doit dans ce cas être changé à la ligne 85 dans le fichier "client.c"
    La durée de chaque phase peut être changée aux lignes 20, 21 et 21 du fichier
"Game" dans le dossier serveur/src/rasenderroboter/
